QUESTIONS

FAQ

Is it really free?

Yes. MIT license. You pay only for the LLM API (like OpenRouter credits) and optional hosting if you want cloud deployment.

What platforms are supported?

9 platforms: LinkedIn (company + personal), Facebook, Instagram, X (Twitter), YouTube Community, Reddit, Google Business, Blog.

How does publishing work?

Social Scheduler sends content to a Make.com webhook. You set up the actual posting in Make.com using their integrations. This avoids dealing with constantly changing social media APIs.

Can I use my own AI model?

Yes. OpenRouter lets you pick from GPT-4, Claude, Llama, Gemini, and others. Bring your API key and choose your model in settings.

Where is data stored?

On your machine in a SQLite file. No cloud sync. Back it up by copying one file. Your content never leaves your computer unless you publish it.

Do I need technical skills?

You should be comfortable with Node.js and npm. It's a developer tool. If you can follow a README and run terminal commands, you're good.
